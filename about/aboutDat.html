About Dat protocol

Dat is a new peer-to-peer protocol. It provides public key addressed file archives which can be synced securely and browsed on demand. It is a decentralized protocol, that is any device can host any archive. Files in dat archive are versioned. Changes are written in an append only log.

Dat is a protocol for sharing data between computers. It's strengths are that data is hosted and distributed by many computers on the network, so that it can work offline or with poor network quality. The original upoader can add or modify data while keeping a full history and it can handle large amounts of data.
